MIN-WIDTH

Pozwala określić nam minimalną szerokość danego obiektu. Min-width nadpisuje "max-width" oraz "width", gdy "min-width" jest większy od "max-width".

Przykładowy tekst

MAX-WIDTH

Pozwala ustawić maksymalną szerokość elementu. Zapobiega zwiększaniu się używanej wartości właściwości szerokość powyżej wartości określonej przez max-width.

Przykładowy tekst

MIN-HEIGHT

Właściwość CSS min-height służy do ustawienia minimalnej wysokości elementu. Określa najmniejszą wysokość, jaką element może mieć, niezależnie od jego zawartości.

Przykładowy tekst

MAX-HEIGHT

Właściwość CSS max-height ustala maksymalną wysokość elementu. Oznacza to, że element nie będzie wyższy niż wartość określona przez max-height, niezależnie od jego zawartości.

Przykładowy tekst

OVERFLOW

Właściwość CSS overflow kontroluje, co się dzieje z zawartością elementu, gdy ta przekracza jego rozmiary (szerokość lub wysokość).
