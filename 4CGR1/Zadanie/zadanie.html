KASKADOWE ARKUSZE STYLÓW-CSS

CSS (Cascading Style Sheets) jest to język, którego używamy do ulepszania I upiększania naszej strony internetowej. Dzięki temu językowi możemy zmienić kolor czcionki, lokalizacje tekstu czy przypisać danemu elementowi strony różne cechy np. Czy po najechaniu na niego ma coś się stać. ​

Sposoby dodania styli do pliku HTML

  • Dodanie w zewnętrznym pliku CSS - Podpinamy plik CSS do pliku HTML za pomocą elemntu "link". Dodajemy element "link" i podajemy źródło
  • Dodanie w pliku HTML - Dodajemy element "style"do elementu "head"
  • Dodanie stylu INLINE - Dodajemy element style do konkretnego selektora​
